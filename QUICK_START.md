# ⚡ БЫСТРЫЙ СТАРТ

## 🎯 За 5 минут запустить бота

### Вариант 1️⃣: На компьютере (локально)

#### Шаг 1: Скачайте Python
https://www.python.org/downloads/ (версия 3.8+)

#### Шаг 2: Клонируйте репозиторий
```bash
git clone https://github.com/YOUR_USERNAME/minon-shop.git
cd minon-shop
```

#### Шаг 3: Установите зависимости
```bash
pip install -r requirements.txt
```

#### Шаг 4: Создайте .env файл
```bash
cp .env.example .env
```

Отредактируйте `.env` и добавьте ваши данные:
```
BOT_TOKEN=your_token_here
ADMIN_ID=your_id_here
API_ID=your_api_id
API_HASH=your_api_hash
TON_WALLET=your_wallet_here
```

**Как получить эти данные?** Смотрите файл `GET_CREDENTIALS.md`

#### Шаг 5: Запустите бота
```bash
python main.py
```

Должно появиться:
```
🚀 Бот запущен...
```

✅ **Готово!** Тестируйте бота в Telegram

### Вариант 2️⃣: На хост-сервисе Railway (Рекомендуется)

#### Шаг 1: GitHub
1. Создайте аккаунт на https://github.com
2. Создайте новый репозиторий (кнопка "New")
3. Назовите его `minon-shop`
4. Не добавляйте README/gitignore
5. Нажмите "Create repository"

#### Шаг 2: Загрузите код на GitHub

**Способ через веб-интерфейс:**
1. Нажмите "uploading an existing file"
2. Загрузите файлы:
   - main.py
   - requirements.txt
   - .env (создайте через Upload)
   - .gitignore
3. Нажмите "Commit changes"

**Способ через Git:**
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/minon-shop.git
git branch -M main
git push -u origin main
```

#### Шаг 3: Railway
1. Перейдите на https://railway.app
2. Нажмите "New Project"
3. Выберите "Deploy from GitHub"
4. Подключите GitHub аккаунт
5. Выберите репозиторий `minon-shop`
6. Railway автоматически начнет деплой

#### Шаг 4: Переменные окружения
На странице проекта в Railway:
1. Нажмите на приложение
2. Перейдите в "Variables"
3. Добавьте переменные из вашего .env:
   ```
   BOT_TOKEN=...
   ADMIN_ID=...
   API_ID=...
   API_HASH=...
   TON_WALLET=...
   ```
4. Нажмите "Save"

#### Шаг 5: Проверка
В разделе "Logs" должно быть:
```
🚀 Бот запущен...
```

✅ **Готово!** Бот работает в облаке 24/7

## 📱 Первые шаги

### 1. Добавьте товар (как админ)

2. Загрузите .session файл

### 2. Выдайте себе баланс (как админ)
1. Нажмите "💎 Выдать баланс"
2. Введите ваш ID
3. Введите сумму (например 100 TON)

2. Выберите купленный аккаунт

## 🆘 Проблемы при запуске?

### Ошибка: "ModuleNotFoundError: No module named 'aiogram'"
```bash
pip install -r requirements.txt
```

### Ошибка: "Invalid token"
Проверьте что BOT_TOKEN правильный в .env файле

### Ошибка: "Cannot connect to Telegram"
- Проверьте интернет подключение
- На некоторых сетях может быть блокировка Telegram
- Используйте VPN если необходимо

### Бот запустился но не отвечает
- Проверьте что токен правильный
- Найдите бота по имени в Telegram
- Отправьте `/start`

### SQLite database is locked
Закройте все другие процессы использующие базу:
```bash
# На Linux/macOS
lsof | grep sifon_market.db

# Удалите старый файл
rm sifon_market.db

# Перезапустите бота
python main.py
```
