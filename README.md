# Minon Shop - Telegram Account Store Bot

🎯 Современный магазин Telegram аккаунтов с реферальной системой

## ✨ Функциональность

### Для пользователей:
- 🛒 Покупка Telegram аккаунтов
- 💰 Пополнение баланса через TON
- 📩 Автоматическое получение кодов входа
- 🤝 Реферальная программа (10% от покупок друга)
- 👤 Профиль с отслеживанием баланса
- 🛍️ История покупок

### Для администратора:
- ➕ Добавление товара (загрузка .session файлов)
- 💎 Выдача баланса пользователям
- 📢 Рассылка всем пользователям
- 📊 Управление каталогом товаров

## 🚀 Установка и запуск

### Требования:
- Python 3.8+
- SQLite3

### Локальная установка:

```bash
# 1. Клонируем репозиторий
git clone https://github.com/YOUR_USERNAME/minon-shop.git
cd minon-shop

# 2. Создаем виртуальное окружение
python -m venv venv
source venv/bin/activate  # Для Linux/macOS
# или
venv\Scripts\activate  # Для Windows

# 3. Устанавливаем зависимости
pip install -r requirements.txt

# 4. Настраиваем .env файл
cp .env.example .env
# Отредактируйте .env и добавьте свои данные

# 5. Запускаем бота
python main.py
```

### Настройка .env файла:

```env
# Bot Token из @BotFather
BOT_TOKEN=YOUR_BOT_TOKEN

# ID администратора
ADMIN_ID=YOUR_TELEGRAM_ID

# Telegram API credentials
API_ID=YOUR_API_ID
API_HASH=YOUR_API_HASH

# TON кошелек для пополнения
TON_WALLET=YOUR_TON_WALLET_ADDRESS
```

### Получение необходимых данных:

1. **BOT_TOKEN**: создайте бота через [@BotFather](https://t.me/botfather)
2. **ADMIN_ID**: ваш Telegram ID (отправьте /start боту и смотрите логи)
3. **API_ID и API_HASH**: получите на [my.telegram.org](https://my.telegram.org)
4. **TON_WALLET**: создайте кошелек на [tonkeeper.com](https://tonkeeper.com)

## 💾 База данных

Бот использует SQLite3. Структура:

- **users** - пользователи и их балансы
- **products** - товары (аккаунты)
- **purchases** - история покупок
- **referrals** - реферальная система

База данных создается автоматически при первом запуске.

## 🔧 Команды для администратора

В главном меню администратора доступны команды:
- **➕ Добавить товар** - добавить аккаунт на продажу
- **💎 Выдать баланс** - начислить баланс пользователю
- **📢 Рассылка** - отправить сообщение всем пользователям

## 📝 Рабочий процесс добавления товара

1. Нажмите "➕ Добавить товар"
2. Загрузите .session файл (получен из Telethon)
3. Укажите цену в TON
4. Укажите страну (локацию)
5. Укажите отлегу (как давно не заходили)
6. Укажите тип (New, Premium, Business и т.д.)

## 🤝 Реферальная система

- Каждый пользователь получает реферальную ссылку
- При покупке через ссылку реферер получает 10% от суммы
- Статистика доступна в разделе "Реферальная ссылка"

## ⚠️ Важные замечания

1. **Безопасность .env файла**:
   - Никогда не коммитьте .env в Git
   - Используйте .gitignore для исключения .env файла
   - На хост-сервисе устанавливайте переменные через панель

2. **Session файлы**:
   - Храните их в защищенном месте
   - Используйте Telethon для получения session файлов
   - Бот проверяет валидность сессии перед продажей

3. **API Telegram**:
   - Используйте реальные API_ID и API_HASH
   - Не используйте тестовые ключи в production

4. **Обработка платежей**:
   - Проверяйте комментарий при пополнении (должен быть ID пользователя)
   - Вручную зачисляйте баланс через "Выдать баланс"
   - В будущем можно интегрировать @CryptoBot для автоматизации

## 🐛 Решение проблем

### Бот не запускается
```bash
# Проверьте версию Python
python --version  # Нужен 3.8+

# Проверьте виртуальное окружение
source venv/bin/activate

# Переустановите зависимости
pip install --upgrade -r requirements.txt
```

### Ошибка при загрузке .session файла
- Убедитесь что папка `sessions/` существует
- Проверьте права доступа
- Убедитесь что диск не переполнен

### Не приходит код входа
- Проверьте что сессия авторизована
- Убедитесь что аккаунт может получать SMS от Telegram
- Попробуйте вручную запросить код на https://web.telegram.org
