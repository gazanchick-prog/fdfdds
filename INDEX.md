# 📚 Документация Minon Shop Bot

## 🎯 Что это?

**Minon Shop** - это полнофункциональный магазин Telegram аккаунтов с поддержкой:
- 🛒 Продажи аккаунтов по разным странам
- 💰 Система баланса в TON
- 🤝 Реферальная программа (10% комиссия)
- 📩 Автоматическое получение кодов входа
- 👥 Админ панель для управления товарами
- 🔐 Безопасное хранение данных

---

## 📖 Как работает?

### Для пользователей:
1. Юзер нажимает `/start`
2. Пополняет баланс (TON или другие способы)
3. Выбирает страну и аккаунт
4. Покупает аккаунт
5. Получает код входа через кнопку "Получить код"
6. Может поделиться реферальной ссылкой и получать 10%

### Для администратора:
1. Добавляет аккаунты через загрузку .session файлов
2. Может выдавать пользователям баланс
3. Может отправлять рассылки всем юзерам
4. Видит все покупки в логах

## ⚡ Быстрые команды

### Локальный запуск
```bash
git clone https://github.com/YOUR_USERNAME/minon-shop.git
cd minon-shop
pip install -r requirements.txt
cp .env.example .env
# Отредактируйте .env
python main.py
```

### Получить session файл
```bash
python get_sessions.py
# Выберите пункт 1 и следуйте инструкциям
```

### Docker запуск
```bash
docker build -t minon-shop .
docker run --env-file .env minon-shop
```

### Deploy на Railway
```bash
git push
# Railway автоматически задеплойт после push на GitHub
```

## 💰 Как заработать?

### Способ 1: Продажа аккаунтов
1. Добавляете аккаунты в магазин
2. Пользователи их покупают
3. Вы получаете деньги в виде TON

### Способ 2: Реферальная программа
- Рефералы получают вашу ссылку
- Когда они покупают, вы получаете 10% от суммы
- Автоматически зачисляется в баланс

### Пример заработка:
- Аккаунт стоит 100 TON
- Реферер получает 10 TON
- При 10 рефералах: 100 TON в месяц

## ❓ FAQ

### Q: Какие нужны API ключи?
**A:** Нужны:
- BOT_TOKEN от @BotFather
- API_ID и API_HASH с my.telegram.org
- TON кошелек адрес
- Опционально: CRYPTOBOT_API_TOKEN для автоматических платежей

### Q: Как получить .session файлы?
**A:** Используйте скрипт `get_sessions.py`:
```bash
python get_sessions.py
```

### Q: На какой сервис залить?
**A:** Рекомендуем Railway.app - самый простой вариант для начинающих.

### Q: Как автоматизировать платежи?
**A:** Интегрируйте CryptoBot - смотрите [PAYMENTS.md](PAYMENTS.md)

### Q: Можно ли использовать PostgreSQL вместо SQLite?
**A:** Да, примеры в [ADVANCED_FEATURES.md](ADVANCED_FEATURES.md)

### Q: Как сделать резервную копию?
**A:** Скопируйте файл `sifon_market.db` или используйте облако.

### Q: Бот не отвечает, что делать?
**A:** 
1. Проверьте что BOT_TOKEN правильный
2. Проверьте интернет подключение
3. Смотрите логи (docker logs / консоль)
4. Перезагрузите бота

## 🚨 Проблемы при запуске?

### Все логи и решения:
```bash
# Ошибка: ModuleNotFoundError
pip install -r requirements.txt

# Ошибка: Invalid token
# Проверьте .env файл

# Ошибка: Cannot connect to Telegram
# Проверьте интернет / используйте VPN

# Ошибка: Database is locked
rm sifon_market.db
python main.py
```

Смотрите **[README.md](README.md)** в разделе "Решение проблем"
